# student-performance-analysis-python
Python-based system to analyze student performance data from CSV files. Uses Pandas and NumPy for data cleaning and calculations, applies pass/fail logic, and visualizes subject-wise averages using Matplotlib. Focused on data preprocessing, automation, and basic analytics.
# Student Performance Analysis System

This project analyzes student academic performance using Python. 
It reads data from CSV files, cleans missing values, calculates average marks,
applies pass/fail logic, and visualizes subject-wise performance.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib

## Features
- Data cleaning and preprocessing
- Average and result calculation
- Subject-wise performance visualization

## How to Run
1. Install required libraries
2. Run analysis.py
3. Run visualization.py
